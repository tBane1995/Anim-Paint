float brighten(float c, float atten, float curve) {
    return c + (1.0 - c) * atten * pow(1.0 - c, curve);
}

//...
float atten = 0.7f;
float curve = 0.95f;

r1 = brighten(r1, atten, curve);
g1 = brighten(g1, atten, curve);
b1 = brighten(b1, atten, curve);


-curve > 1.0 → efekt jeszcze mocniej preferuje ciemne tony
-curve < 1.0 → efekt jest bardziej równomierny
-c to kanał w zakresie 0–1
-(1.0 - c) jest większe dla ciemnych kolorów, więc one dostają większy „zastrzyk” jasności
-jasne kolory mają małe (1.0 - c), więc zmiana jest minimalna
-(1 - c) jest podniesione do potęgi, żeby mieć lepszą kontrolę jak szybko efekt zanika dla jaśniejszych pikseli.
